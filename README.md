# 🛒 TechVerse Market

A modern **Multi-Vendor E-Commerce Platform** built as a semester project using **HTML5, CSS3, Bootstrap, Vanilla JavaScript, and Firebase**. The platform provides separate dashboards for **Admin**, **Vendor**, and **Buyer**, allowing users to buy, sell, and manage products efficiently.

---

# 📖 About the Project

TechVerse Market is a web-based marketplace where multiple vendors can sell their products while buyers can browse and purchase them. The platform is managed by an admin who approves vendors and products before they become available to customers.

The project focuses on creating a clean, responsive, and user-friendly shopping experience using Firebase for authentication, database, and storage.

# 👥 User Workflow

## Buyer

Register/Login → Browse Products → Add to Cart → Checkout → Place Order → Track Orders

## Vendor

Register → Wait for Admin Approval → Login → Add Products → Manage Inventory → View Orders

## Admin

Login → Approve Vendors → Approve Products → Manage Users → Manage Orders → View Reports
